## Supported Versions

This portfolio is a **personal static site** with no production backend, but I still take its security seriously. The following versions receive security updates:

| Version | Supported          | Status |
|---------|--------------------|--------|
| `main`  | ✅ Active          | Latest development branch |
| `gh-pages` | ✅ Active       | Deployed live site |
| Older commits | ❌ No          | Use `main` instead |

> **Note:** Since this is a static site with no server-side code, the attack surface is limited to client-side vulnerabilities (XSS via user-controlled content, malicious dependencies, etc.).

---

## 🐛 Reporting a Vulnerability

I take all vulnerability reports seriously — even for a personal portfolio, the principles matter. If you discover a security issue, please follow responsible disclosure:

### How to Report

**Preferred method:** Use [GitHub Security Advisories](https://github.com/ItsWanheda/portfolio/security/advisories/new) for **private** disclosure.

**Alternative method:** Email me directly at:
* 📧 **Wanheda.work@gmail.com**

### What to Include

Please provide as much detail as possible:

## Vulnerability Summary
[One-sentence description]

## Steps to Reproduce
1. Navigate to ...
2. Click on ...
3. Observe ...

## Expected Behavior
[What should happen]

## Actual Behavior
[What actually happens]
